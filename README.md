<h2> 오승재 </h2>

- 활동 정보저장 **[POST]**
- 스쿱 저장 누를떄마다 Question Card **[POST]**
- Question **[GET]**
- 작성해놓은 Question Card **[GET]**
- 작성해놓은 Question Card Update **[PUT]**
- 활동 클릭시, 활동별 Data **[Get]**
- API 명세서 작성

---

<h2> 임찬기 </h2>

- 회원가입 **[POST]**
- 로그인 **[POST]**
- 해시태그 생성 **[POST]**
- 유저별 활동 Data 전체 **[GET]**
- 즐겨찾기 Update **[PUT]**
- 즐겨찾기 활동 전체 **[GET]**
- 유저별 전체 해시테그 **[GET]**
- 활동 필터링 모아보기 **[GET]**
